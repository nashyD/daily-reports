◆ Your stack

Funding Jun 4–5, 2026

Supabase raises $500M at a $10.5B valuation — and changes the self-hosted Postgres default next week

Supabase announced a $500M Series F at a $10.5B post-money valuation, led by GIC with Stripe (second check) and Salesforce Ventures joining and all existing investors participating — roughly doubling its valuation in eight months and pushing total raised past $1B. The company frames the round around "agentic infrastructure": vibe-coding and AI agents are spinning up databases at a pace it wants to fund.

Two operational notes from the same June window matter more to you than the headline number. First, the default self-hosted Docker Compose database image moves from Postgres 15 to Postgres 17 the week of June 15. Second, since June 3, new free-tier projects on Supabase's default email provider can no longer modify their auth email templates (paid Pro+ projects are unaffected).

So what for TSD: Concrete actions. (1) If you self-host any Supabase via Docker Compose, pin your Postgres image version now so the week-of-June-15 default flip to PG 17 doesn't silently change your DB on the next pull — then test the v15→v17 migration deliberately. Your managed Supabase projects are unaffected. (2) For any new client project that needs branded magic-link / passkey / confirmation emails, either start it on a paid plan or wire a custom SMTP from day one — a new free-tier project can no longer edit those templates, and you don't want to discover that mid-build. The funding itself is good news: the platform you build clients on is well-capitalized and not going anywhere.

Dev tools Jun 2026

GitHub Copilot's desktop app opens to all paid users, with cloud agents and Opus 4.8

GitHub dropped the waitlist on its Copilot desktop application for everyone on Pro, Pro+, Max, Business, and Enterprise. The bigger shift is workflow: from the IDE's agent picker you can now hand a task to a cloud agent that opens a GitHub issue and PR on remote infrastructure while you keep working, and remote control of Copilot sessions is generally available on github.com and mobile. Claude Opus 4.8 is now one of the selectable models.

So what for TSD: You already live in agentic coding (Claude Code), so this isn't a tooling switch — it's competitive context. The "describe a task, get an issue + PR from a cloud agent" pattern is becoming table stakes, which means client-side devs and competitors get the same leverage you have. Your edge stays in judgment, taste, and shipping finished products fast — not in having the agent. Worth a 30-minute look at the cloud-agent flow to see if it speeds up your own backlog grind on the TSD apps.

◆ Security & breaches

Patch Jun 10, 2026

A Defender zero-day lands hours after a record Patch Tuesday

June's Patch Tuesday was the largest on record — roughly 208 CVEs including multiple zero-days (among them a wormable Windows kernel RCE, CVE-2026-45657, and an HTTP/2 "Bomb" DoS, CVE-2026-49160). Within hours a researcher published "RoguePlanet," a new Microsoft Defender zero-day that escalates to SYSTEM via a race condition. Separately, SAP shipped fixes including a critical SAML authentication-bypass (CVE-2026-44748, CVSS 9.9), and ServiceNow warned of an unauthenticated API access incident.

So what for TSD: Make sure your dev Mac/Windows and any Windows VMs took the June updates — the kernel RCE is wormable and needs no user interaction. The reusable lesson for your own apps is in the bug classes: HTTP/2 request amplification and SSRF-via-upgrade are the same patterns that bit Next.js's self-hosted server last month, so keep your apps on patched framework versions and don't expose raw self-hosted servers without a WAF/proxy in front.
